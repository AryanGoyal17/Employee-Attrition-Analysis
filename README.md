# Employee Attrition Analysis: Uncovering Drivers of Workforce Turnover

## 📌 Project Overview
Employee turnover is a massive cost for modern enterprises. This project performs an in-depth Exploratory Data Analysis (EDA) on a synthetic IBM Human Resources dataset to identify the core variables that drive employees to leave a company. 

Rather than just reporting who left, this analysis dives into *why* they left—investigating the relationships between compensation, work-life balance, job satisfaction, and overtime to extract actionable business insights.

## 🎯 Key Objectives
* **Identify Attrition Drivers:** Pinpoint which specific demographic and job-related factors correlate most strongly with employee churn.
* **Statistical Profiling:** Analyze distributions of age, income, and tenure between employees who stayed versus those who left.
* **Pattern Recognition:** Utilize multivariate visualizations to uncover hidden relationships (e.g., How does Job Role combined with OverTime impact attrition?).
* **Actionable Insights:** Translate raw data patterns into logical observations that HR departments could use to improve retention strategies.

## 🛠️ Tech Stack
* **Language:** Python
* **Data Manipulation:** `pandas`, `numpy`
* **Data Visualization:** `matplotlib`, `seaborn`
* **Environment:** Jupyter Notebook / VS Code

## 📊 The Dataset
The data used in this project is the **IBM HR Analytics Employee Attrition & Performance** dataset. 
* **Size:** 1,470 records × 35 features
* **Type:** Fictional/Synthetic (created by IBM data scientists for analytical modeling)
* **Target Variable:** `Attrition` (Yes/No)
